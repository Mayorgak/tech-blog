## Installation


 ```bash
Run npm install to install all dependencies
Run schema.sql in MySQLWorkbench

Run node server.js to start the application
 ```


 ## Dependencies

 * Bcrypt
 * Connect-session 
 * Sequelize 
 * Express 
 * Express-handlebars
 * Mysql2
